# _ZIP-IT_

#### By Nathan Young

## Description

_A zip-code guessing game_

## Setup

_Clone and start MAMP server from inside root folder._
_Navigate to localhost:8888 inside of browser._  

## Technologies Used

_This project makes use of:
AngularJS & FireBase._
